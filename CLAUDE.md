# Encantis

Encantis is a programming language that compiles to WebAssembly.

## Documentation

- **[docs/encantis.md](docs/encantis.md)** - Source of truth for language design and semantics. Covers types, operators, type promotion, slices, arrays, and control flow.

## Project Structure

- `tools/` - Compiler toolchain (lexer, parser, checker, codegen)
- `encantis-ext/` - VS Code extension with language server
- `examples/` - Example programs
- `docs/` - Language documentation
